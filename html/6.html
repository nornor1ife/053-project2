หน่วยการเรียนรู้ที่ 6

การตกแต่งเว็บไซต์เพจด้วยcss (cascading style sheet)

การสร้างเว็บไซต์ในปัจจุบันมีภาษาในการพัฒนาหลัก ๆ 2 ภาษา คือ HTML และ CSS ซึ่ง HTML เปรียบเหมือนการกำหนดโครงสร้างของส่วนต่าง ๆ CSS เป็นเหมือนเสื้อผ้าที่นำมาสวมใส่โครงร่าง ทำให้เว็บไซต์ดูน่าสนใจมากยิ่งขึ้น HTML จะมีโครงสร้างกำหนดไว้ เช่น body, p, h1, a ฯลฯ ส่วน CSS เป็นการกำหนดว่า body, p, h1, a นั้นจะให้มีสีพื้นหลัง ตัวอักษรเป็นอย่างไร ขนาดเท่าใด สามารถเขียนแทรกไว้ภายในแท็กของ HTML ได้เลยหรือเรียกว่า CSS แบบ Inline หรือสามารถ แยกออกมาเก็บเป็นอีกไฟล์หนึ่ง หรืออยู่ที่ส่วนหัวของเว็บไซต์ก็ได้ ซึ่ง CSS มีทั้งหมด 3 ประเภท คือ Inline Style Sheet และ Internal Style Sheet รวมไปถึง แบบที่ 3 คือ External Style Sheet คือการกำหนด Style Sheet ไว้นอกไฟล์หลัก มีชื่อว่า Inline Style Sheet คือ การเขียน Style Sheet ไว้ในบรรทัดของแท็ก โดยเขียนแทรกอยู่ในแท็ก HTML เช่น "< p style="color:red"> " Internal Style Sheet คือ การเขียนกำหนดไว้ภายในแท็ก head ส่วน External Style Sheet คือการเขียนไว้ในอีกไฟล์หนึ่ง แล้วเรียกใช้ใน HTML ด้วยคำสั่ง "< link rel="stylesheet" href="style.css"> " การเขียนคำสั่งใน CSS เริ่มต้นจากการใช้ Selector ซึ่งจะเป็นตัวกำหนดว่าเราจะจัดการแท็กใด หรือ Element ใดบ้าง เช่น h1 คือการกำหนดกับตัวอักษรหัวข้อ ตัวอย่างของ Selector มีหลายประเภท เช่น การกำหนดชื่อแท็กโดยตรง ประเภทนี้ 2 Class Selector คือ .ชื่อคลาส และ ID Selector คือ #ชื่อไอดี ในส่วนของ Selector จะอยู่ในวงเล็บปีกกา {} และมีการกำหนดคุณสมบัติหรือคำสั่งต่าง ๆ ในส่วนของ CSS จะมีการกำหนดหลายคำสั่งไว้ในแท็กเดียวกัน เรียกว่า Compound Style คือ รูปแบบคำสั่งที่มีการเชื่อมโยงกัน เช่น a:link คือ เมื่อยังไม่คลิก a:visited คือ เมื่อลิงก์นั้นถูกคลิกแล้ว a:hover คือ เมื่อนำเมาส์ไปชี้ a:active คือ ขณะนั้นคลิกยังไม่ปล่อย
