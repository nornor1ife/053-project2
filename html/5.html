หน่วยการเรียนรู้ที่ 5

HTML (Hypertext Markup Language) เป็นภาษาในการกำหนดโครงสร้างของเอกสารเว็บเพจ โดยเริ่มต้นตั้งแต่ทศวรรษ 90 จนกระทั่งถึงปัจจุบันพัฒนา จนเป็น HTML5 โดยมีการเพิ่มคุณลักษณะใหม่ ๆ เข้ามาหลายประการ เช่น Audio/Video Support, New Format Input, Semantic Markup Page Layout เป็นต้น ในบริบทของ HTML เป็นการนำองค์ประกอบย่อย ๆ แต่ละอย่างที่เรียกว่า อีลีเมนต์ (Element) และแอททริบิวต์ (Attribute) มาใช้งานร่วมกัน และโครงสร้างพื้นฐานของ HTML5 ก็ยังมีโครงสร้างหลักที่ยึดหลักของ HTML อยู่ด้วยกัน 3 ส่วน ได้แก่ ส่วนประกาศ ส่วนหัวของเว็บเพจ และส่วนของเนื้อหาของเว็บเพจ เพื่อนำมาพัฒนาเว็บไซต์ต่อไป
