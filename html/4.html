หน่วยการเรียนรู้ที่ 4

หลักการออกแบบเว็บไซต์

การออกแบบเว็บไซต์ในปัจจุบันเปลี่ยนแปลงไปอย่างรวดเร็ว แต่สิ่งที่ยังไม่เปลี่ยน คือ คุณสมบัติของเว็บไซต์ ที่ดีคือการวางโครงสร้างอย่างเป็นขั้นตอนทำให้สามารถเข้าใจเนื้อหา ความหมายได้ง่าย เพื่อให้เข้าถึงกลุ่มเป้าหมายได้อย่างมีประสิทธิภาพ ความเป็นมิตรกับผู้ใช้งานเป็นอีกสิ่งหนึ่งที่นักพัฒนาเว็บไซต์ ควรคำนึงถึง โดยเริ่มจากการกำหนดเป้าหมายของเว็บไซต์ว่าต้องการจะนำเสนอเรื่องอะไร แล้วกำหนดโครงสร้าง เนื้อหาภายในเว็บไซต์ว่ามีเนื้อหาอะไรบ้าง จัดลำดับความสำคัญของเนื้อหาให้อยู่ในตำแหน่งที่ผู้ใช้สามารถมองเห็น ได้ชัดเจน ไม่ซับซ้อน มีปุ่มเมนู หรือทางเชื่อมโยงภายในเว็บไซต์เพื่อให้ผู้ใช้สามารถเข้าถึงเนื้อหาส่วนต่าง ๆ ได้ง่าย และรวดเร็ว โดยอาจเรียกหลักการนี้ว่า UI (User Interface) ส่วนสีสัน การจัดวางข้อความ ภาพเคลื่อนไหว ภาพนิ่ง เสียง วิดีโอ เป็นองค์ประกอบที่ทำให้เว็บไซต์น่าสนใจ แต่การเลือกใช้สิ่งเหล่านี้ ต้องคำนึงถึงกลุ่มเป้าหมายด้วยเช่นกัน ปัจจุบันเว็บไซต์ส่วนใหญ่เน้นการออกแบบให้รองรับทุกอุปกรณ์ (Responsive Web Design) ความสามารถในการแสดงผลอัตโนมัติตามขนาดหน้าจอของอุปกรณ์ เช่น คอมพิวเตอร์แบบตั้งโต๊ะ คอมพิวเตอร์พกพา แท็บเล็ต หรือโทรศัพท์มือถือ โดยใช้กรอบการออกแบบที่มีอัตราส่วนที่นิยมใช้กัน เช่น 60:30:10 และรูปแบบการจัดสีจากล้อสีต่าง ๆ เช่น Monochrome, Dyad, Triad, Split Complementary, Analogous, Tetrad ซึ่งแนวคิดการออกแบบเว็บไซต์ให้ตอบสนอง (Responsive Web Design) ให้มีการวางสี แบบ Fluid Grid โดยใช้หน่วยวัดเปอร์เซ็นต์ หรือหน่วย em หรือ % หรือ em ภาพก็สามารถย่อขยายตาม Flexible Images เพื่อให้เหมาะสมกับการแสดงผลของอุปกรณ์ต่าง ๆ เช่น CSS3 Media Queries หรือไฟล์แบบ Meta ในการระบุขนาดของหน้าต่างแสดงผล (Viewport) ของอุปกรณ์แต่ละ screen ของ CSS3 เป็นคำสั่งการแสดงผลหน้าจอที่เหมาะสมในแต่ละอุปกรณ์ โดยจะทำการออกแบบ Mobile First เพื่อให้สามารถรองรับ Mobile User ในปัจจุบันที่มีจำนวนเพิ่มขึ้นมาก
