หน่วยการเรียนรู้ที่ 7

การออกแบบเว็บไซต์และการใช้งาน Bootstrap Framework

สำหรับนักเรียนที่ยังไม่มีพื้นฐานหรือกระบวนการวิธีในการออกแบบเว็บไซต์ การทำความเข้าใจ พื้นฐานก็เป็นสิ่งสำคัญในการออกแบบในการเขียนรหัส Wireframe วาดแบบร่างโครงร่างเว็บไซต์ก็มีความสำคัญ ซึ่งจะเห็นภาพรวมของเว็บไซต์ก่อนลงมือเขียนรหัส การใช้เครื่องมือที่เป็นตัวช่วยในการออกแบบก็มีหลากหลาย เช่น Adobe Photoshop, Webflow, Justinmind, Wix.com จะสามารถใช้เครื่องมือช่วยให้เห็นภาพเว็บไซต์ ก่อนลงมือเขียนโค้ดจริง ซึ่งเครื่องมือเหล่านี้ จะให้บริการในลักษณะ Web Application ที่สามารถเข้าถึง ผ่านเว็บไซต์ และบางโปรแกรมสามารถใช้งานผ่านมือถือได้ นอกจากนี้ Bootstrap Framework มาพัฒนา เป็น Web Prototype ซึ่งเป็นเครื่องมือที่ช่วยอำนวยความสะดวกในการออกแบบและจัดวางองค์ประกอบ ต่าง ๆ ได้ง่าย เช่น การลากวาง (Drag & Drop) ของ Bootstrap ที่มีลักษณะ Components ต่าง ๆ สามารถนำมาใช้งานร่วมกันได้ ซึ่งใช้ในการจัดวางโครงร่างเว็บไซต์ที่เหมือนเป็นเลย์เอาต์ (Layout) มีการกำหนดตำแหน่งขององค์ประกอบพื้นฐาน เช่น Header, Nav, Section, Footer ซึ่งสามารถใช้งาน ร่วมกับการเขียน HTML ได้ทันที โดยใน Wireframe จะมีการกำหนด Container แท็กไว้โดยใช้งานได้ทั้ง < div class="container"> หรือ < div class="container-fluid"> จะใช้งานแบบขนาดกว้างเต็มหน้าจอ หรือตามหน้าจอ HTML ก็ได้ ส่วนประกอบอื่น ๆ ของ Bootstrap ก็คือ Grid System ซึ่ง Bootstrap ได้มีการกำหนด Grid System ขึ้นมา โดยแบ่งคอลัมน์ในเว็บไซต์เป็น 12 ช่อง โดยใช้คำสั่ง เช่น < div class="container"> > row > col ซึ่งการจัดเรียง Grid หรือกริดจะจัดวางคอลัมน์ ตามขนาดหน้าจอ (Responsive Web Design) ดังนี้ คำสั่ง sm = หน้าจอขนาดเล็ก md = หน้าจอขนาดกลาง และ xl = หน้าจอขนาดใหญ่ เป็นต้น
