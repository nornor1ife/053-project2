จุดประสงค์รายวิชา

จุดประสงค์รายวิชา เพื่อให้ เข้าใจเกี่ยวกับกระบวนการและโครงสร้างการทำงานของเว็บไซต์ เข้าใจโครงสร้างและไวยากรณ์ของโปรแกรมภาษาหรือกระบวนการใช้เครื่องมือการสร้างเว็บไซต์ เข้าใจหลักการออกแบบและกำหนดส่วนประกอบที่จำเป็นของเว็บเพจ สามารถติดตั้งโปรแกรมจำลอง Web Server หรืออัปโหลด (Upload) เว็บไซต์ สามารถใช้โปรแกรมภาษาหรือโปรแกรมสำเร็จรูปหรือโปรแกรมระบบ CMS สำหรับการออกแบบและสร้างเว็บไซต์ มีเจตคติและกิจนิสัยที่ดีในการปฏิบัติงานคอมพิวเตอร์ด้วยความละเอียด รอบคอบ และถูกต้อง
